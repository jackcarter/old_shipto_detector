# Submission Checklist

- Generate the latest icons and store assets with `./scripts/generate_assets.py`.
- Load the unpacked extension in Chrome and verify banner, popup, and options flows.
- Run `./scripts/package_extension.sh`.
- Upload the ZIP from `dist/` to the Chrome Web Store dashboard.
- Fill out the Store listing tab using `store/listing.md`.
- Fill out the Privacy tab using `store/privacy.md`.
- Upload the three screenshots, store icon, and promo tile.
- Submit for review.
