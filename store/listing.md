# Chrome Web Store Listing Draft

## Name

Old Address Detector

## Short description

Warns you when one of your saved old shipping addresses appears during checkout.

## Detailed description

Old Address Detector helps you catch outdated shipping addresses before you place an order.

Paste any previous address strings into your watchlist, one per line. When a checkout page, order form, or shipping summary contains one of those saved strings, Old Address Detector shows an on-page warning so you can stop and verify the destination before you buy.

Use Old Address Detector to:

- catch old apartment, office, or family addresses before checkout
- keep a private watchlist of shipping destinations you no longer use
- start with major webmail domains ignored by default so inboxes stay quiet
- ignore additional domains where you do not want warnings
- pause or resume scanning at any time
- manage a plain address-string list from a dedicated settings page

Old Address Detector does not send your address data to a server. Your settings stay inside Chrome storage on your browser.
